# 微信公众号 H5 支付并签约

提示

本文档只是本人公司项目实践总结，不同公司开发逻辑可能有些许不同，但流程基本相同。

微信公众号支付并签约官方文档

H5 获取授权 code

微信公众平台开发——微信授权登录（OAuth2.0）

# 需求描述

微信内部浏览器打开商品详情页面（页面已经配置到公众号），执行购买流程

----> 点击购买按钮进入订单结算页面（此步骤需要考虑微信授权问题）

----> 点击确认支付按钮唤起微信签约并完成支付

----> 支付完成后跳转回原页面，并展示相应的支付结果（成功 or 失败）。

  1. 在商品详情页面，点击 ￥19 购买按钮，通过location.href跳转到订单结算页（并把商品信息传递过去）。

    TIP

    由于是微信公众号支付，订单结算页调微信支付并签约时需要传递 code（用户登录凭证）。故此处从商品详情跳转到订单结算时，采用如下跳转逻辑：

// 原理：先跳转到微信授权，然后微信会重定向回redirect_uri页面，
// 同时在页面url后边新增了一个code参数
if (this.terminal === "wechat") {
    let redirect_uri = base.addBaseParams(
        "http://wx.m.dangdang.com/touch/payVue/auto_pay.html?channelMonthlyStrategyId=" +
            data.id +
            "&vipOriginalPrice=" +
            data.originalPrice / 100 +
            "&shouldPayMoney=" +
            (data.firstMonthPrice
                ? data.firstMonthPrice / 100
                : data.newPrice / 100) +
            "&cId=" +
            this.vip.userInfo.channelId +
            "&strategyGrantStatus=" +
            data.strategyGrantStatus
    );

    location.href =
        "https://open.weixin.qq.com/connect/oauth2/authorize?appid=此处是公司的appid&redirect_uri=" +
        encodeURIComponent(redirect_uri) +
        "&response_type=code&scope=snsapi_base&state=123#wechat_redirect";
}

  1. 订单结算页面初始化时：

    • 获取接收传递过来的商品信息(info)、登录凭证(code)；
    • 用info调接口，创建订单、并获取订单号(orderId)；

  2. 点击订单结算页面的确认支付按钮：

    <!-- vue -->

<script>
data(){
    return{
        // 公众号支付所需参数
        wxWapPayParams:null
    }
},
methods:{
    // 微信公众号支付用到的方法
    weixinJSBridgeCheck() {
        // 判断微信的WeixinJSBridge
        if (typeof WeixinJSBridge == "undefined") {
            if (document.addEventListener) {
                document.addEventListener("WeixinJSBridgeReady",this.onBridgeReady,false);
            }else if (document.attachEvent){
                document.attachEvent('WeixinJSBridgeReady', this.onBridgeReady);
                document.attachEvent('onWeixinJSBridgeReady', this.onBridgeReady);
            }
        } else {
            this.onBridgeReady();
        }
    },
    // 支付sdk准备完成
    onBridgeReady(){
        const _this = this;
        // 触发微信支付
        WeixinJSBridge.invoke(
            'getBrandWCPayRequest',
            {..._this.wxWapPayParams},
            function(res){
                if(res.err_msg == "get_brand_wcpay_request:ok" ) {

                    // 微信返回支付成功，轮询订单接口确定支付结果
                    ...省略

                } else {
                    alert(JSON.stringify(res))
                    //  取消支付或者其他情况
                    //  get_brand_wcpay_request:cancel
                    //  get_brand_wcpay_request:fail
                    // window.location.href = "http://e.dangdang.com/media/h5/ddreader50/vipPayResult-page.html?resultCode=0";
                }
            }
        );
    }
}
</script>
