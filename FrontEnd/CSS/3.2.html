# 两端对齐

TIP

两端对齐可以使用text-align:justify属性来实现两端对齐，但这个属性只对多行有效，如果确定只有一行文字，可以使用text-align-last:justify来配合使用，此属性兼容性一般。

<div class="text-box" v-for="(item,index) in dataList" :key="index">
    <span class="label">{{item.label}}</span>：
    <span class="desc">{{item.desc}}</span>
</div>
<style>
.label{
    /* 必须指定宽度 */
    width: 100px;
    /* 关键代码 */
    display: inline-block;
    text-align:justify;
    text-align-last: justify;
}
</style>

# 竖排文字

TIP

竖排文字的实现很简单，它只需要设置writing-mode属性即可，下面是它属性值的介绍：

  • horizontal-tb：默认值，表示水平排版，从上到下。
  • vertical-lr：表示垂直排版，从左到右。
  • vertical-rl：表示垂直排版，从右到左。

下面是竖排文字的代码：

/* 横排 */
.text-horizontal{
    writing-mode: horizontal-tb;
}
/* 竖排左->右 */
.text-vertical-lr{
    writing-mode: vertical-lr;
}
/* 竖排右->左 */
.text-vertical-rl{
    writing-mode: vertical-rl;
}
