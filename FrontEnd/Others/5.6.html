# Google Cloud - Mac 使用 iterm2 自动 ssh 登录远程服务器

通常情况下，有两种方式登录远程服务器。一种是正常使用 ssh 登录（输入账号密码），另一种是使用 ssh 公钥登录（免输入密码）。本文记录从第一种方式入手。

正常使用 ssh 登录时：

ssh user@host -p port
# 例如 ssh root@192.168.81.68 -p 8888
# 会提示输入密码
user@host's password:

# 客户端 iTerm2 配置

  1. 在~/.ssh/目录新建 shell 文件
vim ~/.ssh/googlecloudshell

copy success
Copy successed
  1. Copy 并编辑内容

用户名、ip 地址、密码三项需要修改。密码就是上一步在服务器上新建的密码。

set user 用户名
set host ip地址
set password 密码
set timeout -1

spawn ssh $user@$host
expect "*assword:*"
send "$password\r"
interact
expect eof

copy success
Copy successed
  1. iTerm2 配置

打开 iTerm2 -> preferences -> Profiles

点击下面“+”号，新建一个 profile。

选择 Command 在输入框中输入:

expect ~/.ssh/googlecloudshell

copy success
Copy successed

大功告成 ~ 以后就可以直接免密码登录啦，当然，这个不是真的免密码登录，只是自动免输入密码而已。
