# VuePress 搭建个人博客

提示

这里引用一个张韵晨大神的工具包:

git clone https://github.com/augustVino/vuepress-devkit.git

直接在此基础上搭建就好，省去了很多的坑...

# 一、 VuePress

VuePress 是尤雨溪（vue.js 框架作者）4月12日发布的一个全新的基于 vue 的静态网站生成器，实际上就是一个 vue 的 spa 应用，内置 webpack，可以用来写文档。详见 VuePress中文网

# 二、开始搭建

# 创建项目文件夹

可以右键手动新建，也可以使用 mkdir 命令新建：

mkdir vuepressDemo

# 全局安装 VuePress

npm install -g vuepress

# 进入 vuepressDemo 文件夹，初始化项目

使用 npm init 或 npm init -y（默认yes）

npm init -y

# 创建文件夹和文件

在 vuepressDemo 文件夹中创建 docs 文件夹，在 docs 中创建 .vuepress 文件夹，在.vuepress中创建 public 文件夹和 config.js 文件，最终项目结构如下所示：

vuepressDemo

# 在 config.js 文件中配置网站标题、描述、主题等信息

module.exports = {
    title: "Vino's Blog",
    description: '前端爬坑之旅',
    head: [ // 注入到当前页面的 HTML <head> 中的标签
        ['link', { rel: 'icon', href: '/logo.jpg' }], // 增加一个自定义的 favicon(网页标签的图标)
    ],
    base: '/', // 这是部署到github相关的配置
    markdown: {
        lineNumbers: true // 代码块显示行号
    },
    themeConfig: {
        nav:[ // 导航栏配置
            { text: 'Home', link: '/' },
            { text: '进阶·博文', link: '/Blog/' },
            { text: '爬坑·总结', link: '/FrontEnd/' },
            {
                text: 'Contact',
                items: [
                    { text: 'GitHub', link: 'https://github.com/augustVino/augustVino.github.io' },
                    { text: 'Weibo', link: 'https://weibo.com/neikeda/profile' }
                ]
            }   
        ],
        sidebar: 'auto', // 侧边栏配置
        sidebarDepth: 2, // 侧边栏显示2级
        lastUpdated: 'Last Updated', // string | boolean
        search: true,
        searchMaxSuggestions: 10,
        displayAllHeaders: false
    }
};

# 三、一些Tips

# 代码块高亮

在 .md 文件中书写代码时，可在 ``` 后增加 js、html、json、bash等格式类型，代码块即可按照指定类型高亮

代码：

``` js
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
```

效果：

export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}

# 支持 PWA

VuePress 默认支持 PWA，配置方法如下：

config.js 文件中增加

head: [ // 注入到当前页面的 HTML <head> 中的标签
  ['link', { rel: 'manifest', href: '/photo.jpg' }],
  ['link', { rel: 'apple-touch-icon', href: '/photo.jpg' }],
],
serviceWorker: true // 是否开启 PWA

Copy successed

# 四、部署上线

# 登陆 Github

打开 github 网站，登陆自己的 github 账号

接着我们新建两个仓库：

# 新建仓库一： USERNAME.github.io （不用克隆到本地）

注意

USERNAME 必须是你 Github 的账号名称，不是你的名字拼音，要保证和Github账号名一模一样！

例如我的 Github 账号名称是 augustVino

那么新建仓库，Repository name 就填写为：augustVino.github.io

这个仓库建好后，不用克隆到本地，内容更新修改都在仓库二中进行。

# 新建仓库二：随便起一个名字，比如：vuepressBlog （克隆到本地）

这个项目是用来开发博客的，以后只需要改这个项目就够了。

将 vuepressDemo 文件夹的内容拷贝到仓库二（vuepressBlog文件夹），并在根目录下创建 deploy.sh 文件，内容如下：

#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run build

# 进入生成的文件夹
cd docs/.vuepress/dist

# 如果是发布到自定义域名
# echo 'www.yourwebsite.com' > CNAME

git init
git add -A
git commit -m 'deploy'

# 如果你想要部署到 https://USERNAME.github.io
git push -f git@github.com:USERNAME/USERNAME.github.io.git master

# 如果发布到 https://USERNAME.github.io/<REPO>  REPO=github上的项目
# git push -f git@github.com:USERNAME/<REPO>.git master:gh-pages

cd -

copy success
Copy successed

# 修改仓库二中的 deploy.sh 发布脚本

把文件中的 USERNAME 改成 Github 账号名，例如我的账号名是 augustVino，那么就可以改为：

# 如果你想要部署到 https://USERNAME.github.io
git push -f git@github.com:augustVino/augustVino.github.io.git master

copy success
Copy successed

这样仓库二和仓库一就建立了关联。

简单说二者的关系是：仓库一负责显示网站内容，我们不需要改动它；日常开发和新增内容，都在仓库二中，并通过 npm run deploy 命令，将代码发布到仓库一。

# 在 package.json 文件夹中添加发布命令

"scripts": {
    "deploy": "bash deploy.sh"
}

copy success
Copy successed

# 👏 大功告成，运行发布命令

npm run deploy

copy success
Copy successed

此时打开 Github Settings 中下面的链接: https://augustVino.github.io/ 即可看到自己的主页啦~

# 五、发布到自己的个人域名

此处默认已经有了一个域名 😎 下面跟着步骤一步步来就好啦~

# 修改仓库二中的 deploy.sh 文件

将仓库二中的 deploy.sh 文件的第 13 行反注释掉，并填上自己的域名，deploy.sh 文件的最终版：

#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run build

# 进入生成的文件夹
cd docs/.vuepress/dist

# 如果是发布到自定义域名
echo 'blog.augustvino.cn' > CNAME

git init
git add -A
git commit -m 'deploy'

# 如果你想要部署到 https://<USERNAME>.github.io
git push -f git@github.com:augustVino/augustVino.github.io.git master

# 如果发布到 https://<USERNAME>.github.io/<REPO>  REPO=github上的项目
# git push -f git@github.com:<USERNAME>/vuepress.git master:gh-pages

cd -


copy success
Copy successed

此时，我们运行 npm run deploy 即可发布到自己的专属域名啦~
