# Sublime Text3快捷键

本文主要是总结Sublime Text3编辑器的快捷键，方便开发。

# 呼出当前文件路径的控制台

使用node,Grunt等等要调出控制台的，不必每次都打开相应的项目路径，然后 Shift + 鼠标右键。这样效率很低，当然如果你连这个都不知道的话。。。

需要在 sublime 编辑器中安装 Terminal 插件。

Tips: 快捷键 Ctrl+Shift+T 呼出当前文件路径的控制台（有些情况会和搜狗输入法冲突，自己设置）

# 代码快速排版

依赖于 HTML/CSS/JS Prettify 插件，这是一款可以用于HTML，CSS，Javascript 的集成排版插件.

Tips:安装完快捷键 ctrl+shift+h 一键格式化代码

# 快速切换文件

Sublime Text 提供了一个非常快速的方式来打开新的文件。只要按下 Ctrl+P 并开始输入你想要打开的文件的名称。一旦找到文件，只需按 enter 键，就可以开始直接输入到该文件了！

Tips: 快捷键 Ctrl+P

# 跳转标记

当你编辑一个大文件时，文件中有一堆方法，按 Ctrl+R 将其全部列出来，使他们更容易找到。开始尝试的输入你想要的，然后按 Enter 就可以快速跳转这个方法了。无论是js中的方法还是html中id都可快速定位。

Tips: 快捷键 Ctrl+R

# 快速选中文本（多行编辑）

当你编辑一个大文件时，文件中有一堆方法，按 Ctrl+R 将其全部列出来，使他们更容易找到。开始尝试的输入你想要的，然后按 Enter 就可以快速跳转这个方法了。无论是js中的方法还是html中id都可快速定位。

ctrl+d: 选中光标所占的文本，继续操作则会选中下一个相同的文本。（多按几下试试）

ctrl+click: 单击想要编辑的每一个地方，都将创建一个光标

ctrl+l：选中整行，继续操作则继续选择下一行，效果和 shift+↓ 效果一样

ctrl+shift+l：先选中多行，再按下快捷键，会在每行行尾插入光标，即可同时编辑这些 行。

ctrl+click: 单击想要编辑的每一个地方，都将创建一个光标

ctrl+click: 单击想要编辑的每一个地方，都将创建一个光标

ctrl+G: 跳转到第几行

ctrl+W: 关闭当前文件

ctrl+shift+W: 关闭所有打开文件

ctrl+shift+Enter: 在当前行前插入新行

ctrl+shift+F: 查找并替换

shift+F11: 开启全屏免打扰模式，只编辑当前文件

Alt+shift+数字: 分屏显示
