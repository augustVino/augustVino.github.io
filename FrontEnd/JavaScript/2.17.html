# audio、video标签在各端的自动播放

背景：

公司年底需求要做个用户年度数据统计（类似于支付宝年度统计、网易云音乐年度报告）。统计页面中有个背景音乐，在初始化确认用户已登录的情况下，需要自动播放。涉及端：APP（android、iphone）、微信小程序、H5。

  1. 桌面版的Safari在2017年的11版本宣布禁掉带有声音的多媒体自动播放功能
  2. 在2018年4月份发布的Chrome 66也正式关掉了声音自动播放

也就是说<audio autopaly></audio>、<video autoplay></video>在桌面版浏览器也将失效。了解具体详情

# 解决方案

# 设置 muted

muted 静音的自动播放是被允许的，故这种情况适用于 video 。如以下场景：

  1. video 不需要声音，重点是在让用户看流畅的动画
  2. 视频网站的 video ，刚进入页面时就开始静音播放，但是会给用户展示一个“打开声音”按钮，让用户选择。
<video muted src="" autoplay loop></video>

# 等待用户的交互

这种方式需要用户的交互（点击触摸滑动等，用户不需要非得点击 audio 或 video 去播放）。

# Android 内嵌 H5

android 4.2 webview添加了是否需要用户手势去触发音视频的播放开关，默认为true(需要用户手势触发)。 将webview的这个开关设置为不需要用户触发：

mWebview.getSettings().setMediaPlaybackRequiresUserGesture(false);

# IOS 内嵌 H5

IOS APP给webview设置mediaPlaybackRequiresUserAction后可以允许自动播放。

# 最终方案

<template>
    <audio v-if="isLogin" :src="musicUrl" autoplay="autoplay" loop controls hidden="true" ref="audio"></audio>
</template>
<script>

export default {
    data(){
        return{
            // 此处是通过公共方法获取终端信息
            terminal: base.terminal
        }
    },
    computed:{
        audio(){
            return this.$refs.audio
        }
    },
    mounted(){
        this.playAudio();
    },
    methods:{
        // music
        playAudio(){
            const _this = this;

            document.addEventListener('DOMContentLoaded', DOMLoaded);

            function DOMLoaded() {
                _this.audio.play().then(res => {
                    // 说明当前浏览器支持自动播放
                    console.log('can play');
                }).catch(err => {
                    console.log("couldn't play")
                    if(_this.terminal === 'wechat' || _this.terminal === 'miniprogram'){
                        function audioAutoPlay() {
                            _this.audio.play();
                            document.addEventListener("WeixinJSBridgeReady", function () {
                                _this.audio.play();
                            }, false);
                            document.removeEventListener('DOMContentLoaded', DOMLoaded);
                        };
                        audioAutoPlay();
                    }else{
                        // 说明当前浏览器不支持自动播放，需要用户交互触发
                        function setAutoPlay() {
                            _this.audio.play();
                            document.removeEventListener('click', setAutoPlay);
                            document.removeEventListener('touchstart', setAutoPlay);
                        };
                        document.addEventListener('click', setAutoPlay);
                        document.addEventListener('touchstart', setAutoPlay);
                        document.removeEventListener('DOMContentLoaded', DOMLoaded);
                    }
                });
            };
        }
    }
}
</script>
